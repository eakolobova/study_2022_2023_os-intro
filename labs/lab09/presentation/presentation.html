Создание презентации

Создание нового файла

  1. Откроем emacs.
  2. Создадим файл lab07.sh
  3. Наберем текст:
1 #!/bin/bash
2 HELL=Hello
3 function hello {
4 LOCAL HELLO=World
5 echo $HELLO
6 }

Редактирование файла

  1. Сохраним файл с помощью комбинации Ctrl-x Ctrl-s (C-x C-s).
  2. Проделем с текстом стандартные процедуры редактирования:
  3. Вырежем одной командой целую строку (С-k).
  4. Вставим эту строку в конец файла (C-y).
  5. Выделим область текста (C-space).
  6. Скопируем область в буфер обмена (M-w).
  7. Вставим область в конец файла.
  8. Вновь выделим эту область и на этот раз вырежем её (C-w).
  9. Отменим последнее действие (C-/).

Редактирование файла

Режим нескольких окон

  1. Поделим фрейм на 4 части: разделим фрейм на два окна по вертикали (C-x 3), а затем каждое из этих окон на две части по горизонтали (C-x 2)
  2. В каждом из четырёх созданных окон откроем новый буфер (файл) и введем несколько строк текста.
  3. Переключимся в режим поиска (C-s) и найдем несколько слов, присутствующих в тексте.
  4. Переключимся между результатами поиска, нажимая C-s.
  5. Выйдем из режима поиска, нажав C-g.
